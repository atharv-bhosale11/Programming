# C Programming Concepts
 
This repository contains C programs ranging from fundamental to advanced programming concepts. The purpose of this repository is to practice C programming, strengthen problem-solving skills, and build a solid foundation in structured and modular programming.

## Topics Covered

### C Fundamentals

* Variables and Data Types
* Input and Output (printf, scanf)
* Operators
* Control Structures (if-else, switch)
* Loops (for, while, do-while)
* Functions
* Command Line Arguments

### Modular Programming

* User-Defined Functions
* Function Prototypes
* Scope of Variables
* Header Files
* Code Documentation Standards

### Arrays and Strings

* One-Dimensional Arrays
* Multi-Dimensional Arrays
* String Handling Functions
* Character Arrays
* String Manipulation Programs

### Pointers

* Pointer Basics
* Pointer Arithmetic
* Pointers with Arrays
* Pointers with Functions
* Pointers with Structures

### Structures and Unions

* Defining Structures
* Nested Structures
* Passing Structures to Functions
* Unions and Memory Sharing

### File Handling

* File Input and Output
* Reading and Writing Files
* File Modes
* Error Handling in File Operations

### Dynamic Memory Management

* malloc, calloc, realloc, free
* Memory Leaks and Best Practices

### Advanced Concepts

* Recursion
* Bitwise Operations
* Preprocessor Directives
* Macros and Constants
* Modular Design with Multiple Source Files
* Algorithm Implementation (Sorting, Searching)
* Data Structures (Linked List, Stack, Queue, Trees)

## Repository Structure

Each file demonstrates a specific C concept with simple examples and explanations through code.

Examples:

* Addition.c  
* EvenOdd.c  
* Palindrome.c  
* Factorial.c  
* ArrayDemo.c  
* PointerDemo.c  
* StructureDemo.c  
* FileHandling.c  

## Learning Goal

The goal of this repository is to build a strong foundation in C programming by practicing concepts through hands-on coding examples, progressing from basic to advanced levels.
