# Java Programming Concepts

This repository contains Java programs covering fundamental to advanced programming concepts. The purpose of this repository is to practice Java programming, object-oriented programming (OOP), functional programming, and problem-solving skills.
 
## Topics Covered

### Java Fundamentals

* Variables and Data Types
* Input and Output (Scanner, System.out)
* Operators
* Control Structures (if-else, switch)
* Loops (for, while, do-while)
* Methods
* Command Line Arguments

### Object-Oriented Programming (OOP)

* Classes and Objects
* Instance Variables
* Static Variables and Methods
* Constructors
* Encapsulation
* Access Specifiers
* Inheritance
  * Single Inheritance
  * Multilevel Inheritance
  * Hierarchical Inheritance
* Polymorphism
  * Method Overloading
  * Method Overriding
* Abstraction
  * Abstract Classes
  * Interfaces

### Exception Handling

* Try-Catch Blocks
* Finally Block
* Multiple Exceptions
* Custom Exceptions

### Advanced Concepts

* Packages
* Access Modifiers
* Inner Classes
* Static Blocks
* Final Keyword
* Wrapper Classes
* Generics
* Collections Framework
  * List, Set, Map
  * ArrayList, LinkedList, HashSet, TreeSet, HashMap
* Streams and Lambda Expressions
* Multithreading
  * Thread Class
  * Runnable Interface
  * Synchronization
* File Handling
* JDBC Basics

### Algorithmic and Problem-Solving Programs

* Prime Number Check
* Palindrome Check
* Factorial Calculation
* Fibonacci Series
* Sorting Algorithms (Bubble, Selection, Insertion)
* Searching Algorithms (Linear, Binary)
* Number Pattern Programs
* String Manipulation Programs

## Repository Structure

Each file demonstrates a specific Java concept with simple examples and explanations through code.

Examples:

* Program1_SumOfDigits.java  
* Program2_Palindrome.java  
* Program3_FindMax.java  
* Program4_FindMin.java  
* Program5_MultiplicationTable.java
